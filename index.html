Рекомендации для ваших целей

Для набора массы

  • Создайте профицит калорий (потребляйте больше, чем тратите)
  • Увеличьте потребление белка до 1.6-2.2 г на кг веса
  • Не забывайте о сложных углеводах для энергии
  • Пейте достаточно воды для гидратации мышц
  • Соблюдайте режим питания (5-6 приемов пищи в день)
  • Сочетайте правильное питание с силовыми тренировками

Для снижения веса

  • Создайте дефицит калорий (потребляйте меньше, чем тратите)
  • Сохраняйте высокое потребление белка для сохранения мышц
  • Ограничьте простые углеводы и добавленные сахара
  • Увеличьте потребление клетчатки для сытости
  • Пейте достаточно воды для ускорения метаболизма
  • Сочетайте диету с кардио и силовыми тренировками
